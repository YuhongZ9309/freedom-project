VPN Tutorial

Setting up and booting your Raspberry Pi

  • Step 1
    Install Raspbian OS on your computer
    Recommend downloading the Lite version.
  • Step 2
    Download balenaEtcher
    We will use this program to flash the OS onto a flash drive.
  • Step 3
    Launcher balenaEtcher
    Drag the Raspbian zip file into Etcher and make sure the target is set to the flash drive. Click "Flash" and wait until it finish flashing.
  • Step 4
    After finishing flashing, the flash drive will now be called “boot”.
    The flash drive would automatically be ejected upon completion. Just remove the flash drive and plug it back in.
  • Step 5
    Create a file called "ssh" with no extension
    You might be prompted to reformat the drive, but DO NOT PRESS REFORMAT. Just ignore it and open the flash drive folder. Create a file called “ssh” with no extension.
  • Step 6
    Booting up your RaspberryPi
    Eject flash drive and plug it into the Raspberry Pi and connect Ethernet. Then, plug in the micro-USB to power it on. Very important to plug in the flash drive before powering.
  • Step 7
    Install PuTTY
    Since we are doing a headless installation, we will use PuTTY to access the SSH terminal over the internet.
  • Step 8
    Open PuTTY
    Open the program, enter raspberrypi in Host Name and press enter. If that does not work, you can also find and enter the IP address of your RaspberryPi instead.
  • Step 9
    Login to your RaspberryPi
    You will be prompted to log into your Raspberry Pi. Login in using the information below.
    Default Username: pi
    Default Password: raspberry
    If you run to the issue “Network Error: Connection Refused” when trying to connect to your RaspberryPi, you did not create the file “ssh” correctly. (step 4)

Installing PiVPN and configuring

  • Step 1
    Copy and paste the installation command into the terminal.
    curl -L https://install.pivpn.io | bash

    The command can be found here.

  • Step 2
    PiVPN Automated Installer
    Once finishing installing, it would said PiVPN would transform your Pi to a VPN server. Press ENTER to continue.
  • Step 3
    Assigning Static IP Address

    Since this will be a server, having a static IP address is important. Press ENTER to continue.

    For more information on what is a static IP address, click here.

  • Step 4
    Internet Configuration

    It will now prompt about your internet configuration to confirm. Here, you will configure the static IP address of your RaspberryPi. When you finish, press ENTER to continue.

    Different router have different ways of setting a static IP address, so please google and search how to set a static IP address for your specific router model.

  • Step 5
    PiVPN User

    You will be prompted about which user to hold the VPN configuration. Press ENTER to continue.

    Since we are doing this installation on Raspberry Pi, there will be only one user to select. Press ENTER to select and continue.

  • Step 6
    Unattended Upgrades

    You will be prompted about whether to enable "Unattended Upgrades" or not. It is highly recommended to select YES. Press ENTER to continue. The RaspberryPi will then some time to processing.

  • Step 7
    Protocol

    You will be prompted about which protocols to uses. It is recommended to select UCP Protocols, and then press ENTER to continue.

    For more information on the difference between UCP and TCP protocols, click here.

  • Step 8
    OpenVPN Port

    You will be prompted about which port the the VPN is going to use. The default port is 1194. Pressed ENTER to continue.

    Then, it will ask you to confirm the port. Press ENTER to confirm and continue.

  • Step 9
    OpenVPN Installation Mode

    You will be prompted about enabling the features of OpenVPN 2.4. Select YES and press ENTER to continue.

    For more information on what is OpenVPN, click here.

  • Step 10
    Encryption Level

    ou will be prompted to choose the level of encryption that you want. 256-bit encryption is recommended, so press ENTER to continue.

    Higher encryption means your connection between your device and the VPN would be more secure. However, it will require the VPN server more performance to decrypt the information. Sometimes, higher encryption can lead to lower speed.

  • Step 11
    Public IP address or Public DNS

    You will be prompted about selecting Public IP address or public DNS. Selecting Public IP address requires you to have a static WAN IP address. However, selecting Public DNS requires you to set up a Public DNS entry. Select your choice and press ENTER to continue.

    For more information on IP address, click here.

  • Step 12
    Configuring DNS Server

    You will be prompted about which DNS server you want to select for your PiVPN. Rrecommended to select Google for DNS due to their speed and security. Then, press ENTER to continue.

    For more information on what is DNS, click here.

  • Step 13
    Installation Complete

    Congratulations! You have successfully installed PiVPN server and almost ready to use. You will be prompted to reboot the system. Select YES and press ENTER twice to confirm reboot.

    PuTTY might display a message about “unexpectedly closed network connection”. Just click OK and close PuTTY.
